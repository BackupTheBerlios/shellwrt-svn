ShellWrt


If ShellWrt ever takes off, the idea is to create a series of scripts to adminster an OpenWrt WDS network.


ShellWrt will use ssh with keys to avoid password and will run on your favorite “shell”, either on an OpenWrt box or your own pc.


Problems so far have been with dropbear which (for me) has not been able to ssh from one OpenWrt to another OpenWrt box with public keys...suk!

OpenSSH has worked between two OpenWrt boxes so I'm dropping dropbear for the time being and compiling RC6 with OpenSSH instead so I can “get on with the job”
